# Logistic Regression - Iris Dataset Classification

This project demonstrates how to solve a **binary classification problem using Logistic Regression** in Machine Learning.  
The model is trained on the famous **Iris dataset** to classify flower species based on their features.

---

## 📌 Project Overview

Logistic Regression is a supervised machine learning algorithm used for **classification problems**.  
In this project, we use Logistic Regression to classify iris flowers into different species based on:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 📊 Dataset

The dataset used in this project is the **Iris dataset**, which contains:

- 150 samples
- 4 input features
- 3 target classes

Target Classes:
- Setosa
- Versicolor
- Virginica

Dataset Source:
```
sklearn.datasets.load_iris()
```
